# 서버 비활성화 해제

마인리스트에서는 일정 기간동안 작동하지 않는 서버는 비활성화된 서버로 간주합니다. 비활성화된 서버일 경우 실시간 서버 상태 확인 기능이 제공되지 않으며, 다시 실시간 서버 상태를 확인하려면 활성화를 해주어야 합니다.

### **서버 비활성화 해제하기**

비활성화 서버인 경우 "> 서버 비활성화가 무엇인가요?"를 클릭하고, 지시에 따르면 서버 재활성화가 가능합니다.
